classic
- Разыскиваемый может ливнуть с сервера 
- Разыскиваемый может пользоваться сундуками

extra
- Разыскиваемый не может ливнуть с сервера (потеряет весь свой инвентарь)
- Разыскиваемый не может пользоваться сундуками, шалкерами и.т.п

hard (+ extrim)
- Разыскиваемый не может есть геплы (Золотые яблоки), хорусы и.т.п (+ я не стал с зельями морочится)
### Разыскиваемый не может использовать перлы (ЧТО ПОШЛО НЕ ПО ПЛАНУ) (Решетки значут что пункт отменён для реализации)

demon (+ hard + extrim)
- Разыскиваемый начнет выживать без ресурсов
- Разыскиваемый получит эффект замедления 2
- ХП разыскиваемого будет в 2× раза меньше
- Голод игрока будет понижен до 7 единиц сытости
